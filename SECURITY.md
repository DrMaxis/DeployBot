# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.x     | ✅        |

## Reporting

Report privately via [GitHub security advisories](https://github.com/DrMaxis/DeployBot/security/advisories/new). Do not file public issues for vulnerabilities.

Acknowledgement within 72 hours. Coordinated disclosure typically targets 30 days.

## Scope

In scope: the `drmaxis/deploybot` package — HMAC verification, admin-allowlist enforcement, webhook URL handling, replay-guard correctness.

Out of scope: host-application code, upstream dependency vulnerabilities (report to those projects), user misconfiguration.
